# ColSON pip

ColSON parser for Python 3.

> ⚠️ **In per-issue development**  
> Any help or critique is encouraged.

## Features

- `colson.loads(x)` parses ColSON data into Python object.
- `colson.dumps(x)` parses Python object into ColSON data.

## Installation

Run:

```bash
pip install colson
```

Import:

```python
import colson
```
